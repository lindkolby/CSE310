# Java Gradebook App

## Overview

This software is a menu-driven Java gradebook application. The program allows a user to add students, enter grades, search for students, remove students, view individual student reports, and view a class report with averages and letter grades.

I wrote this program to demonstrate my understanding of Java as a programming language. The project uses Java syntax, variables, expressions, conditionals, loops, functions/methods, classes, objects, and the `ArrayList` data structure from the Java Collection Framework.

[Software Demo Video](https://youtu.be/iqBPl1c8_2k)

## Development Environment

- Visual Studio Code
- Java Development Kit (JDK)
- Git / GitHub

The programming language used for this project is Java. The main Java library used is `java.util.ArrayList`, which stores students and grades dynamically. I also used `java.util.Scanner` to read user input from the terminal.
