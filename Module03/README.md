# C# Task Scheduler

## Overview

I created this software to demonstrate my understanding of the C# programming language and basic object-oriented programming principles. This program is a console-based task scheduler that allows users to create, view, search, complete, delete, save, and load tasks.

My purpose for writing this software was to practice C# syntax and learn how to organize a program using classes, methods, variables, loops, conditionals, and file input/output. The program also demonstrates the stretch challenge because it reads task data from a file and writes task data to a file.

[Software Demo Video](https://youtu.be/RWCFJPYfQ_8)

## Development Environment

I used Visual Studio Code and the .NET SDK to create this project.

The program was written in C#. It uses standard C# libraries including `System`, `System.Collections.Generic`, `System.Globalization`, `System.IO`, and `System.Linq`.
