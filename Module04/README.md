# NFLVerse Team Performance Analyzer

## Overview

I created this software to demonstrate my understanding of Python, data analysis, and machine learning concepts. This program downloads real NFL play-by-play data from NFLVerse, analyzes team performance, answers questions about the data, creates graphs, and trains a machine learning model.

My purpose for writing this software was to learn how data scientists work with real-world datasets and use programming to identify trends and patterns. The program demonstrates filtering, sorting, aggregation, data conversion, and data visualization techniques. It also includes a machine learning component that predicts team strength based on statistical performance. The stretch challenge is demonstrated through the creation of multiple graphs that visualize the results of the analysis.

[Software Demo Video](https://youtu.be/M_3uhd-XRVg)

## Development Environment

I used Visual Studio Code to create this project.

The program was written in Python. It uses several Python libraries including `pandas`, `matplotlib`, and `scikit-learn` for data analysis, visualization, and machine learning.
